def fracs_add(first, second):
    a, b = first.split('/')
    c, d = second.split('/')
    a, b = int(a), int(d)
    c, d = int(c), int(b)
    x = (a * d + c * b)
    y = (b * d)
    return '{}/{}'.format(x, y)

FizzBuzz

  • Wyświetl liczby od 1 do 100
  • Jeśli liczba jest podzielna przez 3, to zamiast niej wyświetl Fizz
  • Jeśli liczba jest podzielna przez 5, to zamiast niej wyświetl Buzz
  • Jeśli liczba jest podzielna przez 3 i 5, to zamiast niej wyświetl FizzBuzz
